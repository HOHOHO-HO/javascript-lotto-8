# 자바스크립트 로또 미션

## 구현할 기능 목록

- [ ] 로또 구입 금액을 입력받는다.
- [ ] 구입 금액이 1,000원 단위가 아닐 경우 예외를 발생시킨다.
- [ ] 구입 금액에 해당하는 개수만큼 로또를 발행한다.
- [ ] 발행된 로또 번호들을 오름차순으로 정렬하여 출력한다.
- [ ] 당첨 번호 6개와 보너스 번호 1개를 입력받는다.
- [ ] 당첨 번호와 보너스 번호의 유효성을 검사한다 (개수, 중복, 범위 등).
- [ ] 발행된 모든 로또에 대해 당첨 등수를 계산한다.
- [ ] 당첨 통계(등수별 당첨 개수)를 계산하여 출력한다.
- [ ] 총 수익률을 계산하여 소수점 둘째 자리에서 반올림하여 출력한다.
- [ ] 사용자의 잘못된 입력이 있을 경우, 해당 시점부터 다시 입력을 받는다.

## 테스트할 기능 목록

- [ ] **로또(Lotto) 모델**
  - [ ] 로또 번호의 개수가 6개가 아닐 때 예외가 발생하는지 테스트한다.
  - [ ] 로또 번호에 중복된 숫자가 있을 때 예외가 발생하는지 테스트한다.
  - [ ] 로또 번호가 1~45 범위 밖일 때 예외가 발생하는지 테스트한다.
- **애플리케이션 로직**
  - [ ] 구입 금액에 따라 정확한 개수의 로또가 발행되는지 테스트한다.
  - [ ] 각 등수(1등~5등, 꽝)에 대해 당첨 여부가 정확히 계산되는지 테스트한다.
  - [ ] 전체 당첨 통계가 올바르게 집계되는지 테스트한다.
  - [ ] 수익률 계산이 정확한지 테스트한다.
- **예외 처리**
  - [ ] 구입 금액, 당첨 번호, 보너스 번호 입력 시 발생할 수 있는 다양한 예외 상황을 테스트한다.
